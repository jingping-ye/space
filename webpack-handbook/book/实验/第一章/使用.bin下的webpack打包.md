# 实验 01：使用.bin 下的 webpack 打包

直接使用以下命令，就可以将index.js文件打包生成bundle.js文件，直接在index.html中引入bundle.js即可。

```sh
./node_modules/.bin/webpack index.js -o bundle.js
```

```sh
--output, -o                  The output path and file for compilation assets
```
